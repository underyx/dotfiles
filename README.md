# dotfiles [![build status][build status badge]][build status]

## Installation

1. Install `git`, `zsh`, `python`, and the `en_US.UTF-8` locale
2. Run `git clone https://github.com/underyx/dotfiles && dotfiles/install`

Whee! :tada:
